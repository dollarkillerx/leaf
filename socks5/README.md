# SOCKS5 代理服务器

这是一个使用 Rust 和 Tokio 编写的异步 SOCKS5 代理服务器。

## 功能特性

- 支持 SOCKS5 协议
- 异步 I/O 处理
- 支持 IPv4、IPv6 和域名解析
- 无认证模式
- 高性能数据转发

## 构建和运行

### 构建项目

```bash
cargo build --release
```

### 运行服务器

```bash
# 设置日志级别
export RUST_LOG=info

# 运行服务器
cargo run --release
```

服务器默认监听在 `127.0.0.1:1080`。

## 使用方法

### 配置客户端

您可以使用支持 SOCKS5 的客户端连接到代理服务器：

- **代理地址**: 127.0.0.1
- **端口**: 1080
- **协议**: SOCKS5
- **认证**: 无

### 测试连接

您可以使用 curl 来测试代理：

```bash
curl --socks5 127.0.0.1:1080 http://httpbin.org/ip
```

## 协议支持

- ✅ SOCKS5 握手
- ✅ CONNECT 命令
- ✅ IPv4 地址
- ✅ IPv6 地址
- ✅ 域名解析
- ❌ 认证（当前仅支持无认证模式）
- ❌ UDP 转发
- ❌ BIND 命令

## 日志

服务器会输出详细的日志信息，包括：
- 连接建立
- 握手过程
- 目标地址解析
- 数据传输状态

## 性能优化

- 使用 Tokio 异步运行时
- 8KB 缓冲区进行数据转发
- 并发处理多个连接
- 零拷贝数据转发
